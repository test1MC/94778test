Submission

BCS electronically tracks quote and policy application submissions submitted by an agency. It prevents quoting an identical submissions multiples times if you receive the same submissions from different agencies.

When you clear an submission, BCS associates the client name and client ID with that submission. This also creates a history of application submissions for clients with whom you write insurance coverage and for those where coverage was never issued.

Submission versus Application

In BCS, an application is a collection of submissions. Individual submissions are grouped with other submissions to form an application. For example, when you start the process of creating a submission, you create the application information first. This information consists of:

  • The client address
  • An application type
  • Effective date
  • Agency and agent

The next step of the process is where you create the actual submission by entering such things as the underwriting unit and line of business. When you submit it, BCS creates an application and groups the submission or submissions you created in the newly created application. You can also add new submissions to existing applications.

Edit Submission

You can edit an existing submission. Click on the Edit button to make the fields active.

BCS displays a warning message if the agency associated with the submission has been referred to another agency. You can update the agency information from the warning.

Policy Numbers

BCS includes functionality to generate a policy number that you can use when you issue a policy from your policy issuance system. BCS manages those policy numbers behind the scenes, and when you request a policy number through the policy number generator function, BCS automatically selects the next available policy number and assigns it to the current application submission.

You can also add a policy number manually to a submission.

Submission Status

The submission status displays in the upper-portion of the page. Click on the status to view the status history for the submission. The pop up window displays the current status, date the status changed, and who changed the status. It also displays any notes added during the status update.

You can reverse the status of a submission. For example, if you accidentally changed the status from pending to rated, you can reverse the status change back to pending. The Update Status menu contains a Reverse Status item. When you select it, the Reverse Status window displays. You will be able to select a status that you want to revert the submission to. If the list displays several status changes, you can select the status you want to revert to from the list.

If you select either Missed Quote or Declined from the Status Reason list, BCS requires you to also select a reason from the Reason list.

Copy Submission

You can copy the information from one submission to a new submission in BCS. This allows you to reuse the current submission setup for a client without having to re-enter information into the new submission.

What do you want to do?

Edit Submission

Use this procedure to edit a submission.

  1. Search for the submission using the Submission Search feature.
  2. Open the Submission.
  3. Click Edit.
  4. Make your changes to the submission.
  5. Click Save. BCS saves your submission changes.
 Update Submission Status

Use this procedure to update a submission status.

  1. Open the submission in the Submission page.
  2. Click Update Status.
  3. If you selected either Missed Quote or Declined, select a reason from the Reason list.
  4. Enter any additional information about the submission in the Notes field.
  5. Click Update. BCS saves the new status information.
 Copy Submission

Use this procedure to copy a submission to a new submission.

  1. Open the submission you want to copy in the Submission page.
  2. Click the down arrow located next to the Update Status button.
  3. Click Copy Submission.
  4. Indicate on the Duplicates page that this application is not a duplicate and click Continue.
  5. Make the necessary changes on the application page.
  6. Click Submit Application. BCS creates a new submission application for the client.
 Switch Submission to another Client

Use this procedure to switch a submission to another client.

  1. Open the submission you want to switch in the Submission page.
  2. Click the down arrow located next to the Update Status button.
  3. Click Switch Client. The Switch Client window opens.
  4. Review the information in the window to make sure it is the correct submission.
  5. Click Continue.
  6. Enter the client ID for the new client in the New Client ID field and click Continue.
  7. Review the information in the window. If it is correct, click Switch Client. BCS assigns the submission to the new client.
  8. Click Close.
 Delete Submission

Use this procedure to delete a submission.

  1. Open the submission you want to delete in the Submissions page.
  2. Click the down arrow located next to the Update Status button.
  3. Click Delete Submission. A warning pop up displays.
  4. Click OK. BCS changes the status to Deleted.
