Berkley Clearance System

Berkley Clearance System (BCS) is an application used to clear new application submissions. BCS prevents your company from quoting the same submission twice, whether it is submitted multiple times by the same agency or submitted once by different agencies. The Client name and Client ID number is associated with all application submissions for that client. This helps maintain a history of application submissions for each client for whom you write insurance.

When you add a new client to BCS, the client information is also added to a client database. Each application interacting with BCS accesses and uses that information, resulting in time savings and consistency in how the name and address information are entered.

You have three search methods available:

  • Search by client
  • Search by submission number
  • Search by agency and agent

You can use each method to view the submissions associated with the search type. For example, if you search for an agency and agent, you can view all submissions that came from that agency. If you search for a client, and if that client exists, you can view the submissions associated with the client.
