Application

The Application page displays information about an application submitted by a client. It also displays all submissions that are part of the application. For example, if the client submitted an application that contained three submissions, information about all three submissions display on the page.

Submission versus Application

In BCS, an application is a collection of submissions. Individual submissions are grouped with other submissions to form an application. For example, when you start the process of creating a submission, you create the application information first. This information consists of:

  • The client address
  • An application type
  • Effective date
  • Agency and agent

The next step of the process is where you create the actual submission by entering such things as the underwriting unit and line of business. When you submit it, BCS creates an application and groups the submission or submissions you created in the newly created application. You can also add new submissions to existing applications.

You can access the different areas of BCS from this page. Click on the links in the page to go to that area of BCS. For example, if you click on the client name BCS displays the Client page.

You can also manage the application process from this page. You can:

  • Manage existing submissions
  • Copy an application
  • Update the submission status
  • Switch application to another client

What do you want to do?

Manage Submissions

Use this procedure to add an existing submission to an application.

  1. Click the down arrow on the Add New Submission button to display the menu.
  2. Select Manage Submissions.
  3. Click Add Submission.
  4. Enter a submission or policy number in the field.
  5. Click Update.
 Copy an Application

Use this procedure to copy application information to a new application.

  1. Click the down arrow on the Add New Submission button to display the menu.
  2. Select Copy Application.
 Update the Submission Status

Use this procedure to update a submission status.

  1. Click the down arrow on the Add New Submission button.
  2. Select Update Status.
  3. Select the status for the submission from the Status list.
  4. Select a reason from the Reason list if you selected Missed Quote or Declined from the Status list.
  5. Enter a description of the status change in the Notes field.
  6. Click Update Status.
 Switch Application to an Existing Client

Use this procedure to switch an application to another client.

  1. Click the down arrow on the Add New Submission button.
  2. Select Switch Client.
  3. Choose An existing client.
  4. Click Continue.
  5. Enter the client ID that you want to switch the application to in the New Client ID field.
  6. Click Continue.
  7. Review the information in the window to make sure it is correct.
  8. Click Switch Client.
  9. Click Close.
 Switch Application to a New Client

Use this procedure to switch an application to a new client.

  1. Click on the down arrow on the Add new Submission button.
  2. Select Switch Client.
  3. Choose A new client.
  4. Click Continue.
  5. Enter the new client information in the fields provided.
  6. Click Continue.
  7. Review the information in the window do make sure it is correct.
  8. Click Switch Client.
  9. Click Close.
