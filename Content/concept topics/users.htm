Users

The Users page allows you to look up and edit existing users. You can also add new users to the user list.

You must have the proper administrative rights and permissions to access this page.

You can also download a list of active users. When you click on the Download Active User list, BCS saves a .csv file to your computer. You can then open it to view all active users for your company.

User Names

User names are typically entered into BCS using the first initial and last name format. For example, if you have a user whose name is Bert Bloom, you would enter the user name as bbloom.

What do you want to do?

View or Edit an Existing BCS User

Use this procedure to view or edit an existing BCS user.

  1. Enter the user name in the Existing User field. A pop up displays the name.
  2. Click on the name in the pop up dialog.
  3. Make any changes to the information on the Users page.
  4. Click Save User.
 Add a New User

Use this procedure to add a new user.

  1. Click Create New User.
  2. Enter the name of the user in the User Name field.
  3. Select one of the following from the User Type list:
    • Read Only: Gives the user read-only access to the selected roles
    • System administrator: Gives the user all roles
    • User: Gives the user role selected from the Roles list
  4. Select the roles you want to assign to the user from the Roles list. If you selected system administrator as the user type, you do not need to select any roles.
  5. Select the companies the user can view from the Viewable Companies section.
  6. Click Create User.
