Search

BCS displays the Search page when you first enter the application. BCS requires that you search for a client before you can add a client to the BCS database

You can search by:

  • Client
  • Submission information
  • Agency and Agent information

BCS Search Function

When BCS searches for a client using the criteria you enter in the fields, it conducts a separate search for each item you enter in a field. If you enter information in four fields, BCS conducts four separate searches through the database. It then takes the results of those searches, combines the information, ranks it, and displays it in the Search Results page.

For example, if you enter a business name and an address as your client search criteria, BCS conducts a separate search for the business name and a separate search for the address. If it finds matching entries in the database, it combines the results from both searches, ranks them based on the number of returns, and displays the client information in the Search Result page.

If you entered only part of the client information as your search criteria, BCS returns all client information that matches your partial search criteria. You can then select the correct client from the results list. However, in order to find the correct client, enter as much information in the search fields as possible.

You can expand the search fields by clicking the Advanced link in the search field area. When you click the link, BCS displays additional search fields. You can then enter additional search criteria. When you click Search, BCS uses the additional search criteria to refine your search results.

If after you search for a client and it does not display in the Search Result list, you can enter the new client. BCS displays the Create Client button in the Search Results section. Click the button to add a new client to BCS.

Click on the following link for the procedure to add a new client: Add a Client

Client Search

You can search for existing clients using the BCS Client search fields. You can search by:

  • Business name
  • FEIN
  • Phone number
  • Address
  • Client ID

You can enter a partial name in the Name field. BCS will return all the results that contain the information you enter in the field.

Some companies can search by portfolio ID and portfolio name. Click Advanced to display these fields to use in your search criteria.

Submission Search

You can search for existing submissions in BCS. You can search by:

  • Submission or Policy number
  • Date range for submissions
  • Date range for effective date
  • Submission type
  • Submission status
  • Submissions associated with an Underwriting unit
  • Submissions associated with an Underwriter

When you search by any method other than the submission or policy number, BCS returns multiple results. You can sort through the list to find the submission you want to view.

Agency Search

You can search for submissions associated with agencies or agents. You can search by the agency or agent name or assigned number. BCS also allows you to search for agencies by city and state location.

Search for a Client

Use this procedure to search for a client.

  1. Enter the name of the client in the Business Name field.
  2. Enter the FEIN in the FEIN field. When you enter the FEIN, be sure to include the dash between the first two numbers and the remaining numbers. If you enter too many numbers, the search will error.
  3. Enter the client phone number in the Phone Number field.
  4. Enter the address information in the Address fields (address, city, state, zip code)
  5. Click Search. BCS searches for the client and returns the results in the Search Results section.
 Search for a Submission By Policy Number

Use this procedure to search by submission or policy number.

  1. Click the Submission tab.
  2. Enter the submission or policy number in the Submission or Policy Number field.
  3. Click Search. BCS searches for the submission and returns the results in the Search Results section.
 Search for a Submission by Submission or Effective Date

Use this procedure to search by a submission or effective date range.

  1. Click the Submission tab.
  2. Enter a date in the From field.
  3. Enter a date in the To field. If you leave this field blank, BCS returns submissions starting at the From date and going to the current date.
  4. Click Search. BCS searches for the submission and returns the results in the Search Results section.
 Search for a Submission by Type, Status, UW Unit, or Underwriter

Use this procedure to search by the remaining fields.

  1. Do one of the following:
    • Select a value from the Type list.
    • Select a value from the Status list.
    • Select a value from the Underwriting Unit list.
    • Select a value from the Underwriter list.
  2. Click Search. BCS searches for the criteria you selected and returns the results in the Search Results section.
 Search for an Agency

Use this procedure to search for an agency.

  1. Click the Agency and Agent tab .
  2. Enter the agency name in the Name field.
  3. Enter the agency number in the Number field.
  4. Click Search. BCS searches for the agency name and number and returns the results in the Search Results section.
 Search for an Agent

Use this procedure to search for an Agent

  1. Click the Agency and Agent tab.
  2. Enter the agent's first name in the First Name field.
  3. Enter the agent's last name in the Last Name field.
  4. Enter the agent's assigned number in the Number field.
  5. Click Search. BCS searches for the agent information and returns the results in the Search Results section.
 Search by Location for an Agency

Use this procedure to search by location for an agency.

  1. Click the Agency and Agent tab.
  2. Click the Advanced link.
  3. Enter the city name in the City field.
  4. Select a state from the State list.
  5. Click Search. BCS searches for agency that match the location criteria and displays the results in the Search Results section.
