Clients

The Client page displays client information, such as:

  • Contact Information
  • Additional names (includes DBA and Named Insureds)
  • Class codes associated with the client
  • All submissions for the client.

You can click on each tab to view that client information. For example, if you click on the Submissions tab, you can view all submissions associated with the client.

Search for a Client

To access the client page, you must first search for a client to see if it exists in the client database. If BCS finds the client, it returns the client information to the Client Search Results page. If BCS does not find the client, you can enter the new client information by completing the application process.

If you entered only part of the client information as your search criteria, BCS returns all client information that matches your partial search criteria. You can then select the correct client from the results list. However, in order to find the correct client, enter as much information in the Client search fields as possible.BCS conducts a separate search based on the information you enter in each search field. It then takes the results from the searches, ranks them based on the number of time the results are returned, and displays the results.

For example, if you enter a business name and an address as your search criteria, BCS conducts a separate search for the business name and a separate search for the address. It ranks the results based on the number of times the results are returned. It then combines the results from both searches and displays the information in the Client Search Result page.

You can expand the search fields by clicking the Advanced link in the search field area. When you click the link, BCS displays additional search fields. You can then enter additional search criteria. When you click Search, BCS uses the additional search criteria to refine your search results.

 Application Notes

You can add notes to an application. This allows you to add special instructions, contact information, or any other information relevant to the application. When you submit the application, BCS displays the Notes icon next to the submission on the Application page.

 Verify Addresses

When you add an address for a client, BCS can verify the address is valid. BCS uses a Geocode application that checks the address you enter.

Click on the Arrow button next to the Country field. BCS displays a list of addresses similar to the address you entered for the client. If the correct address displays in the list, select that address and click OK. BCS fills the address fields with the correct address.

 Additional Named Insureds and DBA

When you create a new client, you can add DBA and additional named insured information. When you add this information at the client level, it becomes available as selections when you add an application for the client.

Search for a Client

Use this procedure to search for a client.

  1. Click Client in the Search bar.
  2. Enter the client name in the Business Name field.
  3. Enter the client's FEIN number in the FEIN field.
  4. Enter the client's address in the Address fields (Address, city , state)
  5. Click Search. BCS returns the results based on your search criteria.
 Edit Client Information

Use this procedure to edit client information.

  1. Search for the client whose information you want to change. BCS displays the client on the Client Search Results page.
  2. Click on the client name in the Search Results list.
  3. Click Edit. The fields on the page become active.
  4. Make your changes.
  5. Click Save. BCS saves your changes.
 Add a New Client

Use this procedure to add a new client.

  1. Click Create Client on the Search Results page.
  2. Click Add Additional Name to add a new additional named insured or designate the DBA name.
  3. Enter information in the remaining fields on the New Application page.
  4. Click Continue.
  5. Enter the application information and click Continue.
  6. Enter the submission information and click Submit Application. BCS saves the client information to its database.
 Add Application

Use this procedure to add the application information.

  1. Click Create Application on the Client page.
  2. Select an insured name from the Insured Name list, if different from the primary named insured.
  3. Select the DBA name from the Doing Business As list, if necessary.
  4. Check the additional insured name to add to the application from the Additional Names list.
  5. Check the correct address from the Address(es) list, if necessary.
  6. Select an application type from the Type list.
  7. Enter the effective date of the application in the Effective Date field.
  8. Enter an expiration date in the Expiration Date field if different than the default expiration date.
  9. Enter the agency name in the Agency field.
  10. Select the appropriate agent from the Agent list.
  11. Click Continue.

If BCS flags the application as a duplicate, click on the link below for Duplicate information.

 Add Submission Information

Use this procedure to add the submission information.

  1. Click Add Class Code. The Class Code field becomes active
  2. Enter the class code in the field.
  3. Select an underwriting unit from the Underwriting Unit list.
  4. Select an underwriter from the Underwriter list if you want to change the default underwriter.
  5. Select a policy symbol from the Policy Symbol list. This defines the line of business you want insured.
  6. Select the office that will service the policy from the Service Office list.
  7. Enter codes in the three code fields, if necessary.
  8. Select one of the following from the Policy Number section:
    • Generate upon completion: Select this option to have BCS generate a policy number when you submit the application.
    • Enter on my own: Select this option to manually enter a policy number.
  9. Select the appropriate status from the Status list. BCS defaults the status to Pending
  10. Click Submit Application. BCS creates the client and assigns a submission number to the submission.
 Add Application Notes

Use this procedure to add application notes.

  1. Click on the Application Notes icon located beneath the Cancel button.
  2. Enter your notes in the Note Message field.
  3. Click Save Note.
