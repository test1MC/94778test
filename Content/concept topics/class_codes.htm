Class Codes

The Class Codes page allows you to look up and change existing class codes or add new class codes to the system.

You must have the proper administrative rights and permissions to access this page.

When you first access the Class Codes page, it displays a list of all class codes associated with your company. You can select a class code from the list to edit that class code. Click anywhere in the class code line to display the Edit Class Code window.

You can also create a new class code from this page.

BCS displays the class code along with the SIC and NAICS codes. You can then select the business and market segment that you want to associate the class code with.

Edit an Existing Class Code

Use this procedure to view and edit existing class codes.

  1. Click on Class Codes on the Administration tab.
  2. Enter the class code you want to edit in the Class Code field.
  3. Click Lookup.
  4. Click on the class code in the Matching Class Codes list
  5. Make your changes to the class code in the Edit Class Code window.
  6. Click Save Changes.
 Add a New Class Code

Use this procedure to add a new class code.

  1. Click on Class Codes on the Administration tab.
  2. Click Create New Class Code.
  3. Enter the details of the class code in the Details fields.
  4. Select the appropriate business segment item from the Business Segment list.
  5. Select the appropriate market segment item from the Market Segment list.
  6. Enter any hazard grades for the new class code in the Hazard Grades fields.
  7. Click Save Changes.
