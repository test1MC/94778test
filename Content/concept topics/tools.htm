Tools

The Tools tab contains three different views:

  • Advanced Submission Search
  • View Class Codes
  • Policy Number Generator

Advanced Submission Search

The Advanced Submission Search view contains fields that allow you to enter specific criteria to return specific results. For example, if you wanted to view the submissions assigned to an certain underwriter, you can select that underwriter from the underwriter list. When you click Search, BCS searches through the database and returns all submissions assigned to that underwriter.

The following table describes the Advanced Submission Search fields.

Field Name Description
Submission or Policy # Use this field to search by a submission or policy number.
Submitted Date Use these fields to search by a submission date. If you leave one of the fields empty, BCS uses the date as the starting point for the search. For example, if you enter March 1 2015 in From field, BCS returns all submissions submitted on March 1 to the current date. Enter dates in the From and To fields to return submissions submitted for a date range.
Effective Date

Use these fields to search by policy effective date. If you leave one of the fields empty, BCS uses the date as the starting point for the search. For example, if you enter March 1 2015 in From field, BCS returns all submissions with an effective date of March 1 to the current date. Enter dates in the From and To fields to return submissions submitted for a date range.
Expiration Date Use these fields to search by policy expiration date. If you leave one of the fields empty, BCS uses the date as the starting point for the search. For example, if you enter March 1 2015 in From field, BCS returns all submissions with an expiration date of March 1 to the current date. Enter dates in the From and To fields to return submissions submitted for a date range.
Last Updated Use these fields to search by the date a submission was last updated. If you leave one of the fields empty, BCS uses the date as the starting point for the search. For example, if you enter March 1 2015 in From field, BCS returns all submissions last updated on March 1 to the current date. Enter dates in the From and To fields to return submissions submitted for a date range.
Type Select an item from the list to search by submission type.
Status Select an item from the list to search by a submission status.
Agency Use this field to search for submissions associated with an agency.
Underwriting Unit Select an item from the list to search for a submission associated with an underwriting unit.
Underwriter Select an item from the list to search for a submission associated with an underwriter.
Policy Symbol Select an item from the list to search for a submission by policy symbol (line of business).
Secondary Symbol Select an item from the list to search for a submission by a secondary policy symbol.
BLINK Submitted Select an item from the list to search for a submission submitted through BLINK.
Service Office Select an item from the list to search for a submission associated with a service office.
GL Class Code Use this field to search for a submission that has a specific class code.
SIC Code Use this field to search for a submission that has a specific SIC code.
NAICS Code Use this field to search for a submission that has a specific NAICS code.
Prior Carrier Select an item from the list to search for a submission associated with a specific prior carrier.
Prior Carrier Premium Use this field to search for a submission using the prior carrier premium information.

View Class Codes

The View Class Codes view allows you to enter either the class code number or class code description to view the class code information. For example, if you enter a specific class code number, when you click Search BCS displays the complete class code information on the page.

You can also enter a partial description in the field to view multiple class code descriptions. For example, if you enter construction in the search field, when you click Search BCS displays all class codes that contain construction in the class code description.

Policy Number Generator

The Policy Number Generator allows you to generate a policy number. You can then assign the policy number to a new submission. When a user from the policy issuance system enters the submission number and starts a new quote or policy, BCS sends the policy number to that system which then uses the policy number for the quote or policy.

What do you want to do?

Search for a Submission

Use this procedure to conduct an advanced search for a submission.

  1. Click the Tools tab.
  2. Decide the type of search you want to conduct. For example, if you want to view all submissions that were submitted on a specific date, you would enter the date in the Submitted Date fields.
  3. Enter your search criteria in the appropriate fields for your search type.
  4. Click Search. BCS displays the results in the page.
 View Class Codes

Use this procedure to view class codes.

  1. Click the Tools tab.
  2. Click View Class Codes.
  3. Do one of the following:
    • Enter the class code in the field.
    • Enter a description in the field.
  4. Click Search. BCS displays the results in the page.
 Generate a Policy Number

Use this procedure to generate a policy number

  1. Click the Tools tab.
  2. Click Policy Number Generator.
  3. Click Generate Policy Number. BCS displays the new policy number in the field.
